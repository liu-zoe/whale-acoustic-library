Manual labeling test — 7 focused segments

7 high-confidence, high-SNR keep clips, picked for diversity (farthest-point sampling in Perch embedding space). Each is trimmed to its loudest 5-second window — just the call moment, no padding ambient. Goal: see whether manual call-type labeling is feasible at scale.
How to use this page
  1. For each clip, listen to the focused 5 s segment (the actual call moment). If it helps, the full 30 s clip is also linked for context.
  2. Compare against the SFU SRKW call catalogue (linked below) — find the closest Ford code. Type it in the label field (S01, S17ii, etc.). Use unk if you truly can't decide.
  3. Optional note for context. Auto-saves to browser localStorage.
  4. When done, click Export labels at the top — that JSON file is what we'll use to assess whether manual labeling is practical at the full-dataset scale (350 keeps).
