Pilot for a planned multi-site, multi-year project. Audio sourced from Orcasound (CC BY-NC-SA 4.0). Reference recordings from the Watkins Marine Mammal Sound Database. Detection models: OrcaHello (Orcasound), Multispecies Whale Model (Google), Perch 2.0 (Google DeepMind). Cross-referenced with Acartia visual sightings. The reviewer is not a trained bioacoustician — call-type / pod assignments are best-effort hypotheses. See the findings document for full methodology and caveats.
